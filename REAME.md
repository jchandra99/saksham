# Dastak Monitoring

Digital field monitoring tool for Sanchari Rog Niyantran Abhiyan (Dastak Campaign) — April 2026.

## Setup

1. Install Flutter SDK.
2. Run `flutter pub get`.
3. Run `flutter run`.
4. Build release APK with `flutter build apk --release`.
